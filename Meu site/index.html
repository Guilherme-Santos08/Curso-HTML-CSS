Hierarquia de Texto

Como funciona a Hierarquia?

<h1>são para assuntos principais, e é sobre aquele assunto que iremos falar.

<h2> significa que é um sub-assunto do <h1> que está em cima em cima dele.

<h3> significa que esse é um sub-assunto do <h2> que está acima dele. E assim sucessivamente até o <h6>.

Cuidados

Hierarquia de títulos não se trata de tamanho.

Temos que seguir a ordem, primeiro <h1> depois <h2> depois <h3> ...

Negrito e Itálico

Oque eu devo fazer caso queira usar Negrito ou Itálico?

Caso eu queria usar o Negrito eu preciso usar a tag strong.

Ja o itálico eu tenho que usar a tag EM.

Existe mais de uma tag para strong ou itálico?

sim existe mais de uma tag para esses dois, mas não é recomendado usar elas.

O negrito pode ser usado com a tag B.

Ja o itálico pode ser usado como I.

Porque não e recomendado usar essas duas tag?

Não devemos usar essas duas tag, porque elas não tem semântica.
